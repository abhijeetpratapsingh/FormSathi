# How to Make a Passport Photo Under 50KB

![FormSathi Tools workflows](../assets/screenshots/tools-workflows.png)

`Passport photo under 50kb` is one of the highest-intent searches in this category because many forms set a tight size limit for photo uploads. Users usually need a file that is small enough for the portal but still sharp enough to avoid rejection.

## Typical requirements

A portal may ask for:

- file size under `50KB`
- JPG or JPEG format
- exact dimensions such as `200x230`
- plain background
- visible face and shoulders

## Best workflow

1. Start with a sharp, well-lit photo.
2. Keep the face centered and visible.
3. Resize to the required dimensions first.
4. Compress gradually until the file goes below `50KB`.
5. Recheck the final output on the phone before upload.

## Common reasons photos fail

- wrong dimensions
- face too small
- too much background
- file is under 50KB but looks blurry
- portal expects JPG and the file is PNG

## FAQ

### Is under 50KB enough for every portal?

No. Some portals ask for higher or lower limits.

### Should I compress or resize first?

Resize first when exact dimensions are given. Compress after that.

### Can I use a studio photo?

Yes, if it still matches the portal rules on size, background, and crop.
