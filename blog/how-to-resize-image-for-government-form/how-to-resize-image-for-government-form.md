## What government form portals usually ask for

- passport photo dimensions
- signature dimensions
- image file under a specific KB limit
- JPG or JPEG format
- plain background and readable content

## Universal process that works

1. Read the official instructions carefully.
2. Separate the task into dimensions, file size, and format.
3. Crop the image first.
4. Resize to the required dimensions.
5. Compress to match the size limit.
6. Preview the final file before upload.

## FAQ

### What is the difference between resizing and compressing?

Resizing changes width and height. Compressing reduces file size.

### Which should I do first?

If the portal gives exact dimensions, resize first. If it only gives a KB limit, focus on crop and compression.

### Is it safe to use random websites for government form documents?

That depends on the service, but many users prefer offline workflows for privacy-sensitive files.
