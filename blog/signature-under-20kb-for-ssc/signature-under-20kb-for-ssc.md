# Signature Under 20KB for SSC Forms

![FormSathi Tools workflows](../assets/screenshots/tools-workflows.png)

If you are filling an SSC form, one of the most common upload problems is the signature file. Users often search `signature under 20kb for SSC` because the portal rejects files that are too large, too blurry, or prepared in the wrong format.

## What SSC users usually need

SSC forms commonly expect:

- a small signature file
- a clean white background
- readable strokes
- correct format, usually JPG or JPEG

The exact requirement can change between notifications and portals, so the latest SSC instructions always win.

## How to make an SSC signature file that works

1. Create a fresh signature on white paper.
2. Capture it clearly with your phone.
3. Crop out all unnecessary space.
4. Resize only if SSC mentions dimensions.
5. Compress until the file is below `20KB` but still readable.
6. Double-check the preview before upload.

## SSC-specific rejection reasons

- signature too light
- background not plain enough
- uploaded file exceeds the size limit
- wrong format
- signature looks stretched after resizing

## Practical quality tips

- use dark ink
- avoid shadows across the paper
- crop just outside the signature
- avoid screenshots of old files
- do not upscale a small signature image

## FAQ

### Is 20KB always the SSC limit?

Not always. SSC instructions can change by exam cycle and portal.

### Can I use the same signature for SSC and other exams?

Yes, only if the portal requirements match on size, dimensions, and format.

### What if my signature becomes unreadable under 20KB?

Start with a cleaner original image, crop tighter, and reduce blank space before compressing again.
